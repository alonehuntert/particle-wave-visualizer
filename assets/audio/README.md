# Demo Audio Files

Place your demo audio files here:
- `demo1.mp3` - First demo track
- `demo2.mp3` - Second demo track

## Supported Formats
- MP3
- WAV
- OGG

## Notes
Demo tracks are optional. Users can upload their own audio files or use microphone input.
